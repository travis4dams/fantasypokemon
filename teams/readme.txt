----- Introduction -----

The teams folder contains .txt files that contain the PkHex "batch editor" commands that can be used to generate the Pokemon in that team.

See the following links for more information about how PkHex batch editor commands are formatted:
https://github.com/kwsch/PKHeX/issues/516
https://projectpokemon.org/home/forums/topic/45398-using-pkhex-how-to-use-the-batch-editor-in-pkhex/
